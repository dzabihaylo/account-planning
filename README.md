# Grid Dynamics Prospect Intelligence Hub

A local server that serves the account intelligence app and proxies
AI requests to the Anthropic API, keeping your API key server-side.

## Requirements

- Node.js 16 or higher (https://nodejs.org)
- An Anthropic API key (https://console.anthropic.com)

## Setup (one time)

No npm install needed. The server uses only Node.js built-in modules.

## Running

### Mac / Linux

Open Terminal, navigate to this folder, then run:

```
export ANTHROPIC_API_KEY=sk-ant-...your-key-here...
node server.js
```

### Windows

Open Command Prompt, navigate to this folder, then run:

```
set ANTHROPIC_API_KEY=sk-ant-...your-key-here...
node server.js
```

Then open your browser to: http://localhost:3000

## Sharing with colleagues on the same network

Once the server is running, find your machine's local IP address
(System Preferences > Network on Mac, or ipconfig on Windows).
Share that address with colleagues:

  http://192.168.x.x:3000

They can open it in any browser on the same network.
The API key stays on your machine and is never sent to the browser.

## Deploying to the cloud (optional)

The server runs on any platform that supports Node.js:
- Railway (railway.app) - easiest, free tier available
- Render (render.com) - free tier available
- Fly.io - free tier available

On any of these, set ANTHROPIC_API_KEY as an environment variable
in the platform dashboard. No code changes needed.

## Updating the account data

When new account reports are generated by Claude, replace index.html
with the updated file. The server picks it up immediately on the
next page load, no restart required.

## Changing the port

```
PORT=8080 node server.js
```
